# Image Assets

This folder contains all image assets for the portfolio, organized by category.

## Folder Structure

- **projects/** - Screenshots, mockups, and previews for portfolio projects
- **profile/** - Profile pictures and personal images
- **other/** - Miscellaneous images (background textures, icons, etc.)

## Usage

Reference images using: `/images/[category]/[filename]`

Example: `/images/projects/smart-employee-system.png`
